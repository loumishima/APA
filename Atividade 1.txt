Melhores casos
	Insertion sort = O(n)
		Situação: Mesmo o vetor estando ordenado, é necessário checar todos os elementos
	Merge sort = O(n log(n))
		Situação: Ainda que esteja ordenado é necessário dividir a arvore do mesmo jeito
	Quick sort = O(n log(n))
		Situação: O vetor estando ordenado ainda há o custo de comparar o pivô com os demais elementos
	Heap sort = O(n log(n))
		Situação: Ainda é necessário montar a estrutura de heap, mesmo com a ordenação
